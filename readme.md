## About Laravel Chat

The Chat is developed with laravel in a docker environment

## Requirements to install

* docker
* docker-compose

## Installation

> github download
* sudo git clone https://github.com/nmardones/chat2.git

> execute:
* sudo ./bootstrap.sh

> Down docker-compose
* sudo ./down.sh

> Up docker-compose
* sudo ./up.sh
